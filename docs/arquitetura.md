# Arquitetura

## Fluxo Principal

Sessão
↓
Monitoramento de Janelas
↓
Monitoramento de Navegação
↓
OCR Inteligente
↓
Extração de Entidades
↓
Correlação de Eventos
↓
Session Summary
↓
Report Generator

## Componentes

### GUI

Responsável pelo painel do usuário.

### Session Manager

Gerencia abertura e encerramento de sessões.

### Activity Monitor

Monitora continuamente as janelas ativas.

### Browser Monitor

Monitora navegação em navegadores suportados.

### OCR Monitor

Captura texto de aplicações técnicas.

### Entity Extractor

Extrai IPs, ramais e comandos.

### Flow Analyzer

Classifica fluxos de atendimento.

### Session Summary

Consolida os dados da sessão.

### Report Generator

Gera relatórios automáticos.
