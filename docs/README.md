# Assistente de Suporte

## Objetivo

O Assistente de Suporte é uma aplicação desenvolvida em Python para registrar automaticamente atividades executadas por analistas de suporte durante atendimentos técnicos.

O sistema captura informações do ambiente de trabalho, identifica aplicações utilizadas, monitora navegação, registra evidências técnicas, extrai informações relevantes e gera documentação automática dos atendimentos.

## Principais Benefícios

* Redução do tempo gasto com documentação.
* Criação automática de relatórios técnicos.
* Construção de base de conhecimento.
* Rastreabilidade completa dos atendimentos.
* Geração futura de artigos para Wiki.
* Geração futura de conteúdo para LinkedIn.

## Tecnologias Utilizadas

* Python
* SQLite
* EasyOCR
* MSS
* Win32 API
* Tkinter
